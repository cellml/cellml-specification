
.. note::

  A units definition is a conceptual entity representing physical units.

  A units definition is defined by a :code:`units` element or an `:code:import units` element, or is one of the predefined units of table X.

  A units definition has a name, determined by the :code:`name` attribute of its :code:units element or :code:import units element, or by the "Name" column in table X for predefined units.

  A units definition has a unit reduction, determined by the **TODO** rules in section XYZ.
